📝 ขั้นตอนการกรอกแบบฟอร์ม

  1. เข้าสู่ระบบผ่านอีเมล @kmitl.ac.th
  2. คลิก “กรอกแบบฟอร์ม” แล้วเลือกแบบฟอร์มที่ต้องการ
  3. ระบบจะดึงข้อมูลส่วนตัวมาให้โดยอัตโนมัติ
  4. กรอกข้อมูลเฉพาะส่วนที่จำเป็น
  5. กด “ส่งแบบฟอร์ม” เพื่อบันทึก

🔍 วิธีเช็คสถานะเอกสาร

  1. เข้าสู่ระบบ แล้วเลือก “สถานะเอกสาร”
  2. ระบบแสดงสถานะ: รอดำเนินการ / อนุมัติ / ไม่อนุมัติ
  3. คลิกที่ “ชื่อเอกสารที่ต้องการ” เพื่อดูสถานะและรายละเอียดของเอกสาร

✍️ วิธีเปลี่ยนลายเซ็น

  1. เข้าเมนู “โปรไฟล์”
  2. คลิก “แก้ไขลายเซ็น”
  3. เซ็นชื่อใหม่ แล้วกด “บันทึก”
  4. ระบบจะอัปเดตลายเซ็นทุกเอกสารให้อัตโนมัติ
